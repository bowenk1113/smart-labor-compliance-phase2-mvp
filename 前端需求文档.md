# 企业用工与社保合规智能平台 - 前端需求文档

## 一、文档概述

本文档描述企业用工与社保合规智能平台的前端需求，基于 Dify + Python + MySQL 技术架构，首期 MVP 聚焦陕西地区用工与社保合规场景。

## 二、项目背景

### 2.1 项目定位

面向企业 HR、行政、法务、员工服务和合规管理场景的 `区域政策 + 企业制度 + 办事路径` 智能问答与知识运营平台。

### 2.2 首期 MVP 场景

首期选定：**陕西地区企业用工与社保合规知识包**

### 2.3 技术栈

- HTML / CSS / JavaScript
- Vue 3
- Vite 构建工具
- Vue Router
- Axios

### 2.4 技术选型优势

- Vue 3 适合构建问答台、历史记录和管理后台这类状态较多的交互页面，组合式 API 便于复用表格、弹窗、分页、租户切换等逻辑。
- Vite 开发服务器启动快、热更新快，适合 MVP 阶段高频调整页面、文案和交互细节。
- Vue Router 能清晰划分用户端、历史记录、后台登录和后台业务页面，并通过路由元信息承载后台权限控制。
- Axios 统一封装请求拦截器，便于集中处理 JWT、租户编码请求头、401 跳转登录和 API 错误。
- 自研轻量组件比大型 UI 框架更贴合当前后台密集信息展示需求，可减少样式覆盖成本并保持页面一致性。
- 前端内置中英文文案，便于演示、租户后台和后续多语言场景扩展。

## 四、用户端功能需求

### 4.1 智能问答模块

#### 4.1.1 问答输入区

- 文本输入框，支持自然语言提问
- 输入框占位提示文字：如"请输入您的问题，如：陕西产假多少天？"
- 提交按钮，支持回车键提交
- 加载状态显示（调用 Dify 接口时）

#### 4.1.2 推荐问题区

- 显示高频问题或推荐问题
- 可点击快速填充问题
- 推荐问题示例：
  - 陕西产假多少天
  - 西安劳动仲裁去哪里
  - 居民医保断缴后怎么处理
  - 试用期工资标准
  - 加班费计算方式

#### 4.1.3 回答结果区

- 优先给出结论
- 说明适用地区
- 提供办事路径或参考来源
- 结构化展示，支持富文本格式
- 回答区域支持滚动

#### 4.1.4 来源说明区

- 显示政策来源目录
- 文档发布日期
- 适用地区标注
- 知识来源优先选用：
  - 中国政府网
  - 全国人大/官方法规镜像
  - 陕西省人民政府
  - 陕西省人社厅
  - 陕西省医保局
  - 西安市人社局

#### 4.1.5 办事路径卡片区

- 展示相关办事入口
- APP 路径说明
- 仲裁电话、办公地址等信息
- 家庭共济流程说明

### 4.2 历史记录模块

- 用户可查看最近提问记录
- 支持按时间排序
- 支持点击重新查看答案
- 清空历史记录功能

### 4.3 结果反馈模块

- 反馈按钮：有帮助 / 无帮助
- 反馈备注输入框
- 提交反馈功能
- 反馈成功提示

- 按钮hover效果
- 加载状态显示
- 错误提示信息
- 成功操作反馈
- 表单验证提示

## 七、前端 API 接口需求

### 7.1 问答接口

```
POST /api/chat
请求参数：
{
  "question": "用户问题",
  "user_id": "用户ID（可选）"
}
响应：
{
  "answer": "系统回答",
  "sources": ["来源列表"],
  "related_tasks": ["办事路径列表"],
  "response_time": 响应时间
}
```

### 7.2 历史记录接口

```
GET /api/history
请求参数：
{
  "user_id": "用户ID",
  "page": 页码,
  "page_size": 每页条数
}
响应：
{
  "list": [...],
  "total": 总数
}
```

### 7.3 反馈接口

```
POST /api/feedback
请求参数：
{
  "question_id": "问题ID",
  "is_helpful": true/false,
  "remark": "备注"
}
响应：
{
  "success": true
}
```

### 7.4 后台管理接口

- GET /api/admin/logs - 问答日志列表
- GET /api/admin/feedbacks - 反馈列表
- GET /api/admin/faqs - FAQ 列表
- POST /api/admin/faqs - 添加 FAQ
- PUT /api/admin/faqs/:id - 更新 FAQ
- DELETE /api/admin/faqs/:id - 删除 FAQ
- GET /api/admin/sources - 来源列表
- POST /api/admin/sources - 添加来源
- GET /api/admin/knowledge-packages - 知识包列表

## 八、非功能性需求

### 8.1 性能要求

- 页面加载时间 < 3秒
- 问答响应时间 < 5秒（不含 Dify 调用时间）
- 支持高并发访问

### 8.2 兼容性要求

- Chrome 最新版
- Firefox 最新版
- Edge 最新版
- 移动端 Safari（iOS）
- 移动端 Chrome（Android）

### 8.3 安全要求

- XSS 防护
- CSRF 防护
- 输入内容过滤
