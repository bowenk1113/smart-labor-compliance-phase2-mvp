# 试用期常见合规风险提示

官方公开提醒中，试用期的典型风险包括先试用后签劳动合同 试用期过长 重复约定试用期 试用期工资低于法定标准 试用期不缴社保和试用期内随意辞退等。企业如果把试用期当成独立的低成本用工阶段，在劳动争议中往往会处于不利地位。

试用期必须包含在劳动合同期限内，不存在单独的试用期合同。按劳动合同法的一般规则，三个月以上不满一年的合同，试用期不得超过一个月。一年以上不满三年的不得超过两个月。三年以上固定期限和无固定期限合同不得超过六个月。同一用人单位与同一劳动者只能约定一次试用期。

试用期工资受双重底线约束，不得低于本单位相同岗位最低档工资或者劳动合同约定工资的百分之八十，同时不得低于用人单位所在地最低工资标准。试用期内也不能以还没正式录用为由不参加社保。

在解除方面，用人单位只有在能证明劳动者存在不符合录用条件或其他法定解除情形时，才可以依法解除试用期劳动合同，并应向劳动者说明理由。无法证明的，试用期辞退也可能构成违法解除。
