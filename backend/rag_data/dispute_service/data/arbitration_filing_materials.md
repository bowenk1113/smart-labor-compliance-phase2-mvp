# 劳动仲裁立案材料准备清单

劳动争议处理中，仲裁申请通常需要准备仲裁申请书 申请人身份证明 授权委托材料 证据材料和证据清单。证据常见包括劳动合同 录用通知 工资条 银行流水 社保缴费记录 考勤记录 工作证 解除通知 聊天记录 医疗诊断和工伤认定材料等。

对企业人力资源岗位或员工个人来说，越早形成证据清单越好。涉及工资争议时，重点是劳动关系 工资约定 考勤和发放记录。涉及违法解除时，重点是录用条件 考核证据 通知送达和制度公示。涉及工伤或社保争议时，则要补充诊断材料 医疗票据和参保证明。

如果案件发生在西安，建议在正式提交前先根据用工地点联系对应区县或功能区仲裁机构，确认管辖 窗口地址 工作时间和是否需要线上预约，再按要求整理纸质和电子材料。
