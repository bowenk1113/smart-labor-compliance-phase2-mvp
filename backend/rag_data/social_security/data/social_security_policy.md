# 社保医保合规知识

用人单位应依法为职工参加社会保险。新员工建立劳动关系后，应及时办理参保登记，常见口径为自用工之日起三十日内办理社会保险登记。

试用期员工仍属于劳动关系存续期间，不能因为仍在试用期就不缴纳社保。员工出具自愿放弃社保声明，也不能当然免除企业法定参保缴费义务。

陕西2025年度职工基本养老保险、失业保险、工伤保险缴费基数上下限口径为：上限23250元每月，下限4650元每月。回答时应提示年度口径，后续年度需要按陕西省最新通知复核。

2026年度陕西城乡居民医保个人缴费标准为每人每年400元，财政补助标准为每人每年不低于700元。集中缴费期和待遇享受期会影响是否能够报销。

陕西医保家庭共济可通过陕西医保APP或小程序办理。授权人和使用人需要符合基本医保参保条件，绑定后使用人以本人医保码结算。
