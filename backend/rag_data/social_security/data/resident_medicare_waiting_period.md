# 陕西居民医保连续参保与等待期说明

根据陕西省医疗保障局二零二五年十二月发布的居民医保缴费提示，二零二六年度陕西城乡居民基本医疗保险个人缴费标准为每人每年四百元，财政补助标准为每人每年七百元。在集中缴费期内正常缴费的参保人，可按规定享受二零二六年一月一日至十二月三十一日的全年待遇。

对不符合连续参保缴费条件的一般参保人员，在集中缴费期内参保缴费的，通常设置三个月固定待遇等待期，待遇享受期自二零二六年四月一日起计算。集中缴费期外参保的，还要严格执行长效机制等待期，等待期内发生的医疗费用一般不予报销。

陕西还对连续参保设置了激励机制。自二零二五年起，连续参保满四年后每连续参保一年，大病保险最高支付限额提高不低于三千元。当年未使用医保基金报销的，次年大病保险最高支付限额也提高不低于三千元。断保后，前期连续参保年限会被清零，但已经形成的奖励额度可以保留。

对未连续参保人员，除固定等待期外，每断缴一年还会增加一个月变动等待期。连续断缴四年及以上的，修复后仍需至少等待六个月。企业在员工入离职转换 灵活就业身份变更和居民医保续费提醒等环节中，应尽量避免形成断缴。
