# 陕西职工医保家庭共济办理要点

陕西省医疗保障局明确，职工基本医保个人账户家庭共济可授权给已参保的父母 配偶和子女使用，用于支付其在定点医药机构发生的个人负担医疗费用。二零二四年五月起，陕西已支持省内跨统筹区家庭共济，不再要求授权人与使用人必须在同一统筹区参保。

线上办理路径通常为陕西医保应用或小程序首页中的家庭共济绑定。办理时需要阅读告知书，填写授权人和使用人信息，包括姓名 证件号 参保地 手机号 开始日期 结束日期以及支付顺位，并完成电子承诺签名后提交。

在使用环节，被共济人使用本人医保码结算。对老人 儿童等没有个人手机设备的情形，可以通过国家医保服务平台应用或支付宝 微信小程序的亲情账户完成功能替代。授权人还可以在陕西医保应用中查询共济消费记录。

陕西现行规则下，一名职工可以同时绑定多位家庭成员。同一个参保人可以接受两位近亲属的共济。企业在员工福利宣导或异地就医答疑中，可以重点提示谁能绑定 是否支持跨市使用 支付顺位怎么设和一二三九三咨询热线等高频问题。
