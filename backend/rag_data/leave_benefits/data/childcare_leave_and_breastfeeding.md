# 陕西育儿假与哺乳期保护要点

陕西实行父母育儿假制度。符合政策生育或者依法收养子女的，在子女三周岁以内，每年给予父母双方各累计十天育儿假。这里的各累计十天意味着父母双方分别享有，而不是家庭总共十天。

条例同时明确，职工在婚假 产假 护理假和育儿假期间按出勤对待，享受相应的工资 福利待遇。企业不能将育儿假作为事假处理，也不宜在绩效 晋级或福利发放上作不利扣减。

对哺乳期保护，女职工生育孩子满一周岁前，所在单位应依照国家和本省有关规定保证哺乳时间并提供哺乳条件。单位因特殊情况确实无法保障的，经协商可给予三到六个月哺乳假。哺乳假期间比照生育津贴标准发给津贴，并计算工龄。
