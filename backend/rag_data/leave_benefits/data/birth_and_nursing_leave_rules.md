# 陕西产假 护理假和三孩增假规则

陕西省人口与计划生育条例明确，职工符合政策生育子女的，在国家规定产假的基础上增加产假六十天，同时给予男方护理假十五天。夫妻异地居住的，男方护理假增加到二十天。女职工参加孕前检查的，在上述基础上再增加产假十天。

对三孩情形，陕西还规定女职工在前述产假基础上再增加十五天产假，男方在护理假基础上再增加十天护理假。企业在员工手册和生育流程里，应将是否三孩 是否异地居住 是否参加孕前检查作为休假核算的重要判断点。

此外，单位若不落实婚假 产假 护理假 育儿假等优待规定，可能被县级以上人力资源社会保障部门责令改正并依法处理。对企业来说，这类假期不是内部福利选择题，而是法定合规要求。
